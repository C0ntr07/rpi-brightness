## Install:


```
git clone https://github.com/WestleyK/rpi-brightness.git
cd rpi-brightness/
make
sudo make install
```

<br>
<br>

## Updating:


```
cd rpi-brightness/
sudo make update
```
*Note:* This will install after updating. <br>

<br>
<br>

## Uninstalling:

```
cd rpi-brightness
sudo make uninstall
```
