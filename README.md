## A simple raspberry pi backlight brightness cli adjuster

Designed and tested for raspberry pi with official 7 inch touchdcreen. <br>

<br>

## Install and uninstall

To install: <br>


```
make
sudo make install
```

<br>

To uninstall: <br>

```
sudo make uninstall
```
